
Angular CLI มีคำสั่ง ng ที่ใช้สำหรับสร้างและจัดการแอปพลิเคชัน
 Angular คำสั่ง ng มีประเภทต่างๆ มากมาย ดังนี้

ng new ใช้สำหรับสร้างโปรเจ็กต์ Angular ใหม่
ng generate ใช้สำหรับสร้างส่วนประกอบ (component) 
โมดูล (module) บริการ (service) และอื่นๆ
ng serve ใช้สำหรับรันแอปพลิเคชัน Angular
ng build ใช้สำหรับสร้างแอปพลิเคชัน Angular
 เป็นไฟล์ที่พร้อมสำหรับการนำไปเผยแพร่
ng test ใช้สำหรับทดสอบแอปพลิเคชัน Angular
ng e2e ใช้สำหรับทดสอบแอปพลิเคชัน Angular แบบ end-to-end
นอกจากนี้ Angular CLI ยังมีคำสั่งอื่นๆ 
อีกมากมายที่สามารถใช้สำหรับจัดการแอปพลิเคชัน Angular ได้อย่างมีประสิทธิภาพ

ต่อไปนี้เป็นตัวอย่างการใช้งานคำสั่ง ng ประเภทต่างๆ

สร้างโปรเจ็กต์ Angular ใหม่
ng new my-app
คำสั่งนี้จะสร้างโปรเจ็กต์ Angular ใหม่ชื่อ my-app

สร้างส่วนประกอบใหม่
ng generate component my-component
คำสั่งนี้จะสร้างส่วนประกอบใหม่ชื่อ my-component

รันแอปพลิเคชัน Angular
ng serve
คำสั่งนี้จะรันแอปพลิเคชัน Angular ในโหมดการพัฒนา

สร้างแอปพลิเคชัน Angular เป็นไฟล์ที่พร้อมสำหรับการนำไปเผยแพร่
ng build
คำสั่งนี้จะสร้างแอปพลิเคชัน Angular เป็นไฟล์ที่พร้อมสำหรับการนำไปเผยแพร่
 เช่น ไฟล์ .js ไฟล์ .css และไฟล์ .html

ทดสอบแอปพลิเคชัน Angular
ng test
คำสั่งนี้จะทดสอบแอปพลิเคชัน Angular ด้วย Karma และ Jasmine

ทดสอบแอปพลิเคชัน Angular แบบ end-to-end
ng e2e
คำสั่งนี้จะทดสอบแอปพลิเคชัน Angular 
แบบ end-to-end ด้วย Protractor
