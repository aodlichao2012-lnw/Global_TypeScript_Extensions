
Directive เป็นกลไกที่ช่วยให้นักพัฒนาสามารถเพิ่มคุณสมบัติหรือพฤติกรรมให้กับองค์ประกอบ HTML 
ได้  Directive มีสองประเภทหลักๆ คือ:

Attribute directive เพิ่มคุณสมบัติให้กับองค์ประกอบ HTML
Structural directive เพิ่มหรือลบองค์ประกอบ HTML
การใช้ directive ช่วยให้นักพัฒนาสามารถเขียนโค้ดที่สั้นและกระชับขึ้นได้
 นอกจากนี้ยังช่วยให้นักพัฒนาสามารถแบ่งโครงสร้างโค้ดออกเป็นส่วนๆ ได้

ตัวอย่างการใช้ Attribute directive

<input type="text" [(ngModel)]="name">

directive ngModel เพิ่มคุณสมบัติ ngModelให้กับองค์ประกอบ input
 คุณสมบัติ ngModel ใช้ในการผูกมัดค่าของอินพุตแบบฟอร์มกับคุณสมบัติ name
  ของส่วนประกอบ

ตัวอย่างการใช้ Structural directive

<div *ngIf="isLoggedIn">
  <h1>Welcome!</h1>
</div>

directive ngIf เพิ่มหรือลบองค์ประกอบ HTML 
ขึ้นอยู่กับค่าของคุณสมบัติ isLoggedIn หากค่าของ
 isLoggedIn เป็น true องค์ประกอบ <div> จะปรากฏขึ้น
  หากค่าของ isLoggedIn เป็น false องค์ประกอบ <div> จะหายไป

Directive สามารถใช้งานได้หลากหลายวิธี
 ตัวอย่างการใช้งาน directive บางส่วน ได้แก่:


เพิ่มคุณสมบัติหรือพฤติกรรมให้กับองค์ประกอบ HTML
ผูกมัดค่าขององค์ประกอบ HTML กับคุณสมบัติของส่วนประกอบ
ตรวจสอบค่าขององค์ประกอบ HTML
เพิ่มหรือลบองค์ประกอบ HTML
การใช้ directive จะช่วยให้คุณเขียนโค้ด Angular ที่มีประสิทธิภาพมากขึ้น
