ngIf เป็น directive ใน Angular 
ที่ใช้สำหรับเพิ่มหรือลบองค์ประกอบ HTML ขึ้นอยู่กับค่าของ expression

รูปแบบการใช้ ngIf มีดังนี้:

<div *ngIf="expression">
  // แสดงองค์ประกอบ HTML นี้หาก expression เป็น true
</div>

ตัวอย่างเช่น

<div *ngIf="isLoggedIn">
  <h1>Welcome!</h1>
</div>


โค้ดด้านบนจะแสดงองค์ประกอบ <div> ที่มีข้อความ "Welcome!" 
ากคุณสมบัติ isLoggedIn เป็น true หากคุณสมบัติ isLoggedIn
 เป็น false องค์ประกอบ <div> จะหายไป

expression ของ ngIf สามารถเป็น expression
 ใดก็ได้ที่คืนค่าค่าแบบ Boolean ค่าของ expression
  จะถูกประเมินเมื่อ Angular โหลดหน้าเว็บ ค่าของ expression จะถูกประเมินอีกครั้งเมื่อ Angular ตรวจพบการเปลี่ยนแปลงใดๆ ใน DOM หรือในคุณสมบัติของส่วนประกอบ

หาก expression มีค่าเป็น true องค์ประกอบ HTML 
จะถูกเพิ่มลงใน DOM หาก expression มีค่าเป็น false
 องค์ประกอบ HTML จะถูกลบออกจาก DOM

ngIf เป็น directive ที่มีประโยชน์สำหรับแสดงหรือซ่อนองค์ประกอบ
 HTML ขึ้นอยู่กับเงื่อนไขบางอย่าง ตัวอย่างเช่น คุณสามารถใช้ ngIf
  เพื่อแสดงหรือซ่อนเมนู แสดงหรือซ่อนข้อความแจ้ง หรือแสดงหรือซ่อนฟอร์ม

ตัวอย่างการใช้งาน ngIf อื่นๆ มีดังนี้:

แสดงหรือซ่อนรายการขึ้นอยู่กับค่าของคุณสมบัติ
แสดงหรือซ่อนองค์ประกอบ HTML ขึ้นอยู่กับค่าของตัวแปร
แสดงหรือซ่อนองค์ประกอบ HTML ขึ้นอยู่กับผลลัพธ์ของฟังก์ชัน
การใช้ ngIf จะช่วยให้คุณเขียนโค้ด Angular ที่มีประสิทธิภาพมากขึ้น
