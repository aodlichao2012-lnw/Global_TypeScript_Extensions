
TypeScript มีวิธีการวนซ้ำ (loop) หลายวิธี วิธีการวนซ้ำที่ได้รับความนิยมมากที่สุด ได้แก่:

การใช้คำสั่ง for

คำสั่ง for สามารถใช้เพื่อวนซ้ำจำนวนรอบที่แน่นอน

รูปแบบการใช้คำสั่ง for มีดังนี้:

for (let i = 0; i < 10; i++) {
  // ทำงานซ้ำ 10 รอบ
}

โค้ดด้านบนจะวนซ้ำ 10 รอบ ในแต่ละรอบค่าของตัวแปร i จะเพิ่มขึ้นทีละ 1

ตัวอย่างการใช้งานคำสั่ง for

for (let i = 0; i < names.length; i++) {
  console.log(names[i]);
}

โค้ดด้านบนจะวนซ้ำรายการ names และแสดงแต่ละชื่อบนคอนโซล

การใช้คำสั่ง for-of

คำสั่ง for-of สามารถใช้เพื่อวนซ้ำรายการ (array) หรือวัตถุ (object)

รูปแบบการใช้คำสั่ง for-of มีดังนี้:

for (const item of items) {
  // ทำงานซ้ำสำหรับแต่ละรายการในรายการ items
}

โค้ดด้านบนจะวนซ้ำรายการ items และดำเนินการกับแต่ละรายการ

ตัวอย่างการใช้งานคำสั่ง for-of

for (const name of names) {
  console.log(name);
}

โค้ดด้านบนจะวนซ้ำรายการ names และแสดงแต่ละชื่อบนคอนโซล

การใช้คำสั่ง while

คำสั่ง while สามารถใช้เพื่อวนซ้ำตราบใดที่เงื่อนไขเป็นจริง

รูปแบบการใช้คำสั่ง while มีดังนี้:

let i = 0;
while (i < 10) {
  // ทำงานซ้ำตราบใดที่ i น้อยกว่า 10
  i++;
}

โค้ดด้านบนจะวนซ้ำตราบใดที่ค่าของตัวแปร i น้อยกว่า 10

ตัวอย่างการใช้งานคำสั่ง while

let i = 0;
while (i < names.length) {
  console.log(names[i]);
  i++;
}

โค้ดด้านบนจะวนซ้ำรายการ names และแสดงแต่ละชื่อบนคอนโซลตราบใดที่ค่าของตัวแปร i น้อยกว่าความยาวของรายการ names

การใช้คำสั่ง do-while

คำสั่ง do-while สามารถใช้เพื่อวนซ้ำอย่างน้อยหนึ่งรอบ เงื่อนไขจะถูกประเมินหลังจากดำเนินการลูป

รูปแบบการใช้คำสั่ง do-while มีดังนี้:
let i = 0;
do {
  // ทำงานซ้ำอย่างน้อยหนึ่งรอบ
  i++;
} while (i < 10);

โค้ดด้านบนจะวนซ้ำอย่างน้อยหนึ่งรอบ ค่าของตัวแปร i จะเพิ่มขึ้นทีละ 1 ในแต่ละรอบ และเงื่อนไขจะถูกประเมินหลังจากดำเนินการลูป

ตัวอย่างการใช้งานคำสั่ง do-while

let i = 0;
do {
  console.log(names[i]);
  i++;
} while (i < names.length);

โค้ดด้านบนจะวนซ้ำรายการ names และแสดงแต่ละชื่อบนคอนโซลอย่างน้อยหนึ่งรอบ
