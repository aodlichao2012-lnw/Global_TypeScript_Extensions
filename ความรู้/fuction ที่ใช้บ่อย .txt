Class
Component - คลาสพื้นฐานสำหรับการสร้างคอมโพเนนต์ Angular
Directive - คลาสที่ใช้เพื่อเพิ่มคุณสมบัติหรือพฤติกรรมให้กับองค์ประกอบ HTML
Service - คลาสที่ใช้เพื่อจัดการงานหรือข้อมูลทั่วไป
Pipe - คลาสที่ใช้เพื่อแปลงข้อมูล
Module - คลาสที่ใช้เพื่อจัดระเบียบคอมโพเนนต์และโมดูลอื่นๆ
Function
ngOnInit() - ฟังก์ชันที่เรียกเมื่อคอมโพเนนต์ถูกโหลดครั้งแรก
ngOnChanges() - ฟังก์ชันที่เรียกเมื่อค่าของคุณสมบัติคอมโพเนนต์เปลี่ยนแปลง
ngOnDestroy() - ฟังก์ชันที่เรียกเมื่อคอมโพเนนต์ถูกทำลาย
ngDoCheck() - ฟังก์ชันที่เรียกเพื่อตรวจสอบการเปลี่ยนแปลงของคอมโพเนนต์
ngAfterViewInit() - ฟังก์ชันที่เรียกเมื่อองค์ประกอบที่มองเห็นได้ของคอมโพเนนต์ถูกโหลดครั้งแรก
ngAfterViewChecked() - ฟังก์ชันที่เรียกเมื่อองค์ประกอบที่มองเห็นได้ของคอมโพเนนต์ถูกตรวจสอบการเปลี่ยนแปลง
ngAfterContentInit() - ฟังก์ชันที่เรียกเมื่อเนื้อหาของคอมโพเนนต์ถูกโหลดครั้งแรก
ngAfterContentChecked() - ฟังก์ชันที่เรียกเมื่อเนื้อหาของคอมโพเนนต์ถูกตรวจสอบการเปลี่ยนแปลง
